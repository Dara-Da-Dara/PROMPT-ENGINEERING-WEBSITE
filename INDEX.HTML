Chain-of-Thought (CoT)

Breaking complex reasoning into step-by-step logical steps improves LLM accuracy by 20-50%.

Zero-Shot CoT

Add "Let's think step by step" to any prompt

Think step by step before answering: [Your question]
Few-Shot CoT

Provide 2-3 examples with reasoning

Q: [Example 1]\nThinking: Step 1... Step 2...\nA: [Answer]

CoT Prompt Builder

Use Cases

  • Multi-step math problems and logic puzzles
  • Algorithm design and RL policy optimization
  • Code debugging and architecture decisions
  • Complex reasoning in MARL scenarios

ReAct (Reason + Act)

Combines reasoning traces with actions, observing results to solve tasks requiring tool use.

ReAct Cycle
Thought: [reasoning] Action: [tool call] Observation: [result] Thought: [next reasoning]

ReAct Prompt Builder

Key Components

  • Thought: LLM reasons about the problem
  • Action: Select and execute a tool
  • Observation: Receive tool output
  • Loop until task is complete

Tree-of-Thoughts (ToT)

Explores multiple reasoning paths, evaluates and prunes branches for complex problem-solving.

ToT Process
1. Generate multiple possible thoughts 2. Evaluate each thought 3. Prune low-quality branches 4. Expand most promising paths 5. Backtrack if needed

ToT Prompt Builder

Advantages

  • Handles non-linear problem spaces effectively
  • Explores multiple solution paths simultaneously
  • Intelligent pruning saves computation
  • Ideal for creative and strategic tasks

RAG (Retrieval-Augmented Generation)

Augment LLM prompts with retrieved external documents for grounded, factual responses.

RAG Flow
1. Retrieve relevant documents 2. Augment prompt with context 3. Generate response based on docs 4. Cite sources

RAG Prompt Builder

RAG Benefits

  • Reduces hallucinations with grounded facts
  • Enables custom knowledge integration
  • Supports source citation and traceability
  • Ideal for research, documentation, and compliance
